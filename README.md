# psp_m17
PSP-3000 M17 reflector client. 

## Building
Run the provided `build.sh` script.

> [!WARNING]
> Make sure to change your callsign before compiling the code if you want to connect to a reflector.

## Running on a PSP-3000
Create a `psp_m17` folder under the `/PSP/GAME` directory. Put the `build/EBOOT.PBP` file there. 

> [!NOTE]
> The appication requires WiFi access - make sure you have it configured and enabled.

## Prerequisites
This project uses [PSPDev](https://github.com/pspdev).
